## I. MỤC ĐÍCH

Tài liệu này quy định **TRÌNH TỰ XỬ LÝ BẮT BUỘC**  
cho mọi API nghiệp vụ trong hệ thống CTVT3.

Mục tiêu:
1. Đảm bảo **đúng nghiệp vụ – đúng kế toán**
2. Ngăn việc:
   - API tự tính số
   - ghi chéo bảng
   - sinh số sai thời điểm
3. Tạo một **khuôn xử lý thống nhất** cho toàn hệ thống

👉 Dev viết API **phải đi đúng luồng này**.
